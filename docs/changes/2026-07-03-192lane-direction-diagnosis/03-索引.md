# 03-索引：192lane 方向诊断

## 模块树

- [[96-vs-192-diagnosis]]：96lane 与 192lane 的静态架构差异诊断
  - 输入：`variants/srad/ours_96lane/`、`variants/srad/ours_192lane/`
  - 输出：是否继续把 96 复制扩成 192 的方向判断

## 横向依赖总表

| 模块 | 依赖 | 说明 |
|---|---|---|
| 96-vs-192-diagnosis | `aie/Config.h` | lane、row block、AIE placement 单一事实来源 |
| 96-vs-192-diagnosis | `aie/ProcessGraph/StencilCoreGraph.h` | output buffer vs pktstream/pktmerge 差异 |
| 96-vs-192-diagnosis | `pl/TopPL.cpp` | TopPL stream 接口、内部 FIFO、packet 拆包 |
| 96-vs-192-diagnosis | `pl/Q0Ctrl.cpp` | q0 stat fan-in / broadcast fan-out |
| 96-vs-192-diagnosis | `conn.cfg` | CU 数、stream 数、depth、Vivado directive |
| 96-vs-192-diagnosis | `Makefile` | 构建目标、依赖、v++ link 参数 |
