# 架构文档：固定化开发与调试流程

## 组件

1. `.agents/skills/srad-ours-192lane-workflow/`：项目总入口 skill。
2. `.agents/skills/superpowers-*`：项目本地 Superpowers 流程副本，覆盖 TDD、debug、verification、plans、execution。
3. `docs/CODEMAP_ours_192lane.md`：避免反复读取长文件的代码地图。
4. `docs/DEBUG_PLAYBOOK.md`：Vitis/AIE/HLS/PS 故障定位流程。
5. `docs/DEVELOPMENT_WORKFLOW.md`：日常开发闭环。
6. `scripts/check_ours_192lane.py`：共享验证命令。

## 数据流

用户需求 → 本地 skill 路由 → code map 定位上下文 → RED 测试 → 最小实现 → GREEN 验证 → journal evidence。
